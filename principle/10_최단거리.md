# 최단거리

## 1. 다익스트라

### 특징

- O((V+E)LogV)의 시간복잡도 by using heap queue

## 2. 벨만포드

### 특징

- 음의가중치의 간선이 존재할 때 최단거리를 찾을 수 있음.

- 단순히 최단거리 뿐 아니라 음수의 사이클이 존재하는지도 알 수 있다.

  

## 3. 플로이드 워샬

### 특징

- DP를 활용하여 최단경로를 구하는 알고리즘
- O(V^3)시간복잡도

- ```
  for k in range(v):
  	for i in range(v):
  		if k == i:
  			continue
  		for j in range(v):
  			if i == j:
  				continue
  			
  			
  ```
